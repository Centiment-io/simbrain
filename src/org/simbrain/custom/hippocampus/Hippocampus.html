Hippocampus model

Based on Alvarez and Squire (1994)

Training

Train the model on cortical patterns by pressing the Train button for patterns 1-4. (Each button press corresponds to training the network on that pattern once).

We suggest training each pattern twice. To to do this, click the Train buttons in this sequence:

  • Pattern 1
  • Pattern 2
  • Pattern 3
  • Pattern 4
  • Pattern 1
  • Pattern 2
  • Pattern 3
  • Pattern 4

Testing

  1. Turn off learning.
  2. Test Patterns 1-4. This corresponds to recall right after learning in the non-amnesia case. Left cortex is activated, and the same pattern should occur in the right cortex. There should be no errors if the network was trained as above.

Other Tests

  1. Lesion hippocampus and test again. This models recall in the amnesia case. Errors are likely to occur. Left cortex will be activated but the same pattern may not occur in the right cortex.
  2. Conslidate. To do this, first unlesion the hippocampus and turn learning on. Then press the consolidate button some number of times. This is like time passing between learning and recall.
  3. Turn learning off and test patterns 1-4. Do this with or without the hippocampus lesioned. You should get different error rates. Compare the results with the two forgetting curves in Alvarez and Squire Fig. 2.
